Detailed Description

This is a PIN tool that traces every floating-point arithmetic operation and displays the arguments and results. This tool can also replace floating-point instructions with arbitrary code to test different floating-point implementations.

Note
All floating-point values are printed as 8 digit hex numbers padded with 0's.
