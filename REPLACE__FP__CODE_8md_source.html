1 Replacing floating-point instructions
2 =====================================
3 
4 When compiling the tool with code to replace floating-point instructions, the type signature of the replacement function should be:
5 
6  FLT32 <replacement_fn>(FLT32, FLT32, OPCODE)
7 
8 The first two function arguments are the two operands to the instruction and the last argument is the opcode of the instruction being replaced. The file containing the replacement function must also include pin.H as a header file.
9 
10 See `replace_fp_ins_simple.cpp` and `replace_fp_ins_complex.cpp` for examples of replacement functions.
