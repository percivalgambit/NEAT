§ IsFpInstruction()

BOOL ftrace::IsFpInstruction ( const INS &  ins)

Return true if an instruction is an SSE floating-point arithmetic instruction. This function is called on every instruction while the instrumented program is running if the KnobFpSelectorName flag is supplied on the command line.

Parameters
[in]insThe instruction to test.
Returns
Whether an instruction is a floating-point arithmetic instruction.
Note
Currently, the instructions defined as SSE floating-point arithmetic operations are:
  • ADDSS
  • SUBSS
  • MULSS
  • DIVSS
